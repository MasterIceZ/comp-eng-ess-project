## Development

### Frontend

1. Edit `public/scripts/config.js`

2. Change directory to `frontend`

```
cd frontend
```

3. Install dependencies

```
npm install
```

4. Run development server

```
npm run dev
```

### Backend

1. Copy `.env.template` to `.env` and change config(s)

2. Change directory to `backend`

```
cd backend
```

3. Install dependencies

```
npm install
```

4. Run development server

```
npm run dev
```
